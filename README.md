
# Ostad Module - 2

This project is for Module-2 tasks of Ostad.app


## Run Locally

Clone the project

```bash
  git clone https://github.com/MahmudZq/Ostad-Module-2
```

Go to the project directory

```bash
  cd Ostad-Module-2
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
